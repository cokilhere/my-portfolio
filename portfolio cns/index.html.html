05 / Synthesis

Learning Reflection


As a dedicated player of games like Mobile Legends: Bang Bang, I deeply understand the importance of seamless, secure multiplayer experiences. Building immersive environments is only half the battle; ensuring those environments remain unexploited and fair for the player base is critical.

Working on large-scale interactive modules like C.U.R.E. required rigorous collaboration. It taught me how to translate technical constraints into clear workflows for 3D artists and UI/UX designers, balancing creative vision with backend logic.

My primary strength is my agility; I can pivot from designing sleek UI/UX in Figma to writing C# backend logic without missing a beat. My current weakness is a lack of enterprise network security experience, but I am actively mitigating this vulnerability through my AWS and CompTIA certification roadmap.

My immediate learning curve involves bridging my knowledge of game engines with backend data protection and network diagnostic tools like Wireshark, alongside pursuing formal certifications.
